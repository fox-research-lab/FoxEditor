# FoxEditor 🦊

A simple vi-like text editor written in Python.

## Features

- View mode and Edit mode
- Black background with orange text
- Red border in edit mode
- Green border in view mode
- Keyboard-based text editing

## Requirements

- Python 3.x
- curses (Windows may require windows-curses)

## Installation

```bash
pip install windows-curses
python foxeditor.py sample.txt
